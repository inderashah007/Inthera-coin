# **Inthera Coin (ITC)** - Menghasilkan Mata Wang Kripto dari Data Buangan Internet

![Inthera Coin](https://yourimageurl.com/logo.png)

**Inthera Coin (ITC)** adalah sebuah projek inovatif yang bertujuan untuk **menukar data buangan internet** menjadi mata wang kripto yang bernilai. Dengan perkembangan pesat dalam dunia digital, jumlah data yang dihasilkan setiap hari semakin meningkat. Kami memanfaatkan data yang tidak digunakan ini untuk menjana **Inthera Coin (ITC)**, yang boleh digunakan atau diperdagangkan oleh pengguna di dalam ekosistem blockchain.

---

## **🎯 Matlamat Projek**

Projek **Inthera Coin (ITC)** bertujuan untuk:

1. **Mengumpul Data Buangan Internet**: Data yang tidak digunakan oleh pengguna, termasuk data lebihan kuota internet, diambil dan diproses.
2. **Menukarkan Data kepada ITC**: Setiap gigabit data yang dibuang akan ditukarkan kepada **Inthera Coin (ITC)**, mata wang kripto yang boleh digunakan atau diperdagangkan.
3. **Menyediakan Platform Blockchain**: ITC beroperasi di atas platform blockchain yang selamat dan boleh dipercayai, memberikan pengguna kawalan penuh terhadap aset mereka.
4. **Memberikan Hadiah kepada Pengguna**: Pengguna akan mendapat ganjaran dalam bentuk ITC apabila mereka menyumbang data mereka untuk lombong.

---

## **🛠 Teknologi yang Digunakan**

Projek ini menggunakan pelbagai teknologi untuk memastikan ia berfungsi dengan lancar dan cekap:

- **CUDA C++**: Untuk pemprosesan data menggunakan GPU (pemprosesan paralel) yang meningkatkan kelajuan pengiraan.
- **Solidity**: Untuk menulis kontrak pintar yang menguruskan mata wang kripto **Inthera Coin (ITC)**.
- **Ethereum Blockchain**: Platform yang digunakan untuk melaksanakan kontrak pintar dan memastikan integriti transaksi.
- **Web3.py**: Perpustakaan Python untuk berinteraksi dengan rangkaian Ethereum dan kontrak pintar.
- **Flask**: Framework Python untuk membina API yang menghubungkan backend dan frontend.
- **HTML/CSS/JavaScript**: Untuk membina antara muka pengguna yang mesra pengguna.

---

## **🚀 Ciri-ciri Utama**

1. **Pengumpulan Data Buangan Internet**:
    - Mengumpul data penggunaan internet yang tidak digunakan dari pengguna.
    - Data dihantar secara automatik dan diproses dalam masa nyata.

2. **Mata Wang Kripto - Inthera Coin (ITC)**:
    - Setiap 1GB data buangan ditukarkan kepada **1 ITC**.
    - Pengguna boleh menyimpan, mentransfer, atau menukar ITC melalui platform blockchain.

3. **Kontrak Pintar untuk Urusniaga**:
    - Kontrak pintar menguruskan keseluruhan transaksi ITC dengan selamat.
    - ITC boleh ditukar atau dipindahkan antara pengguna melalui blockchain.

4. **Antaramuka Pengguna yang Mesra**:
    - Antaramuka web membolehkan pengguna mengakses statistik mereka, melihat ITC yang diperolehi, dan berinteraksi dengan sistem.
    - Pengguna juga boleh membuat transaksi dan menukarkan ITC mereka melalui antaramuka ini.
